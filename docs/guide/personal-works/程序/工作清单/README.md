# toDoList开发文档

## 0.前言

本工具十分常见，几乎各种教学视频都会有相关的案例进行开发。

虽然总体技术难度不难，但是还是要认真对待。

原本从个人的角度来说是不太需要这个工具的，但是随着学习内容的不断增加，日程安排的不断复杂，该工具的重要性已经不是备忘录之类的可以替代的了——即使我有着一般严重的备忘录记录症……

因此，该工具的开发被提了上来。

一个能够记录更多细节和更清晰更详细的待办事项列表
